# rachitpatel.github.io
This repo is for my personal website https://rachitpatel.github.io using Jekyll/Github Pages with a remote theme https://github.com/mmistakes/jekyll-theme-basically-basic.

Once you have all the prerequisites to run Jekyll locally, to run this:

Locally (Dev build with server):  **bundle exec jekyll serve --verbose** (This is for Windows, on Mac/Linux, you may not need 'bundle exec')

Navigate to http://localhost:4000 

Prod build: **bundle exec jekyll build** (Same comment about OS applies here)

You will see "_site" folder with the artifacts.
